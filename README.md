# Autoplugin
Autoplugin for PSVITA
This tool allows you to install the following plugins with one click:
- DownloadEnabler
- NoNpDrm
- NoPsmDrm
- Shellbat
- Vitabright (only 3.60 and PSVITA)
- pngshot
- Kuio by Rinnegatamante
- PSV-VSH MENU by joel16
- Vflux by Applelo
- Repatch by dots-tb
- NoAVLS by SilicaAndPina
